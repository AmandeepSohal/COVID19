# COVID19
Querying data from Covid19 data to gather interesting information by utilizing SQL and Microsoft SQL Server Management Studio 2019. (Data set's data is from 8 Jan 2020 to 23 June 2023)

If you want to download this data set, go to https://ourworldindata.org/covid-deaths and download the data set by scrolling down and selecting your start and end dates as 8 Jan 2020 - 23 June 2023

Apologies for any human errors caused by me in my SQL queries. This is a project to display my skills for SQL queries, using commands such as SELECT, FROM, WHERE, LIKE, CAST, ROUND, JOIN.
